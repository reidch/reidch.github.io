## Project overview

Phoenix is a custom rendering project focused on understanding the full real-time graphics pipeline rather than treating an engine as a black box.

## Highlights

- GGX / Smith / Schlick physically based shading
- Rayleigh and Mie atmospheric scattering
- Shadow mapping and instanced rendering
- Bézier-driven object motion
